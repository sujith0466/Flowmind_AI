# Agent System

FlowMind AI is organized around specialized agents:

- Summarizer Agent: extracts the core meaning from messy input.
- Task Generator Agent: turns context into actionable tasks.
- Action Planner Agent: creates prioritized next steps.
- Workflow Organizer Agent: structures output into a usable workflow.
- Critic Agent: optionally improves clarity, usefulness, and consistency.

The MVP should make agent stages visible enough for users and judges to understand the system.
